Atalhos de Teclado

Navegar Cartões

←↓ / J↑ / K→

Pressione as teclas de seta para selecionar cartões adjacentes em um quadro. Pressione j para selecionar o cartão abaixo do atual. Pressione k para selecionar o cartão acima do atual.

Navegar pelos versos de cartões

← / K→ / J

Pressione as teclas de seta para a esquerda e para a direita para abrir cartões adjacentes em uma lista. Pressione "j" ou a seta para a direita para abrir o cartão abaixo do atual. Pressione "k" ou a seta para a esquerda para abrir o cartão acima do atual.

Abrir/fechar barra lateral

[

Pressione "[" para abrir e fechar o menu da barra lateral de navegação de quadros.

Alternar Menu do Quadro

W / ]

Pressione "w" ou "]" para recolher ou expandir o menu do quadro, a barra lateral à direita.

Abrir Menu Quadros de Cabeçalho

B

Pressione "b" para pesquisar quadros rapidamente. Navegue pelos resultados da pesquisa com as setas para cima e para baixo. Pressione Enter com um quadro selecionado para abri-lo.

Focalizar Caixa de Pesquisa

/

Pressione / para posicionar o cursor na caixa de pesquisa, no cabeçalho.

Arquivar Cartão

C

Pressione c para arquivar um cartão.

Data de Entrega

D

Pressione d para abrir um seletor de data de entrega para um cartão.

Adicionar checklist

-

Pressione "-" para adicionar uma checklist a um cartão.

Anexar link

Command/ControlV

Ao visualizar um cartão, anexe um link copiado usando "Control/Command + V".

Modo de Edição Rápida

E

Se você passar o mouse sobre um cartão e pressionar “e”, o modo de edição rápida será aberto e permitirá a edição rápida do título e de outros atributos do cartão.

Fechar Menu/Cancelar Edição

Esc

Pressione Esc para fechar uma caixa de diálogo ou janela pop-up aberta ou cancelar as edições e os comentários que você está escrevendo.

Salvar texto

Command/ControlEnter

Pressionar as teclas Control + Enter (Windows) ou Command + Enter (Mac) salvará o texto que está sendo escrito. Isto funciona ao escrever e editar comentários, ao editar o título do cartão, o título da lista, a descrição e outras coisas.

Abrir Cartão

Enter

Pressione Enter para abrir o cartão selecionado no momento. Pressione Shift+Enter ao enviar um cartão para abri-lo imediatamente após sua criação.

Abrir Menu Filtro de Cartão

F

Use f para abrir o menu de filtro do cartão. A entrada da pesquisa por título é focalizada automaticamente.

Etiqueta

L

Pressione l para abrir uma janela pop-up das etiquetas disponíveis. Clique em uma etiqueta para adicioná-la ou removê-la do cartão.

Alternar Nomes de Etiquetas

;

Pressionar a tecla de ponto e vírgula ";" mostra ou oculta os nomes das etiquetas em um quadro. Você também pode clicar em qualquer etiqueta de um quadro para alternar.

Adicionar/Remover Membros

M

Pressione m para abrir o menu Adicionar/Remover Membros. Clique no avatar de um membro para atribuí-lo ou cancelar sua atribuição.

Inserir Novo Cartão

N

Pressionar "n" abre uma janela pop-up que permite adicionar um cartão depois do que está selecionado, ou em uma lista vazia.

Criar novo cartão com texto

Command/ControlV

Quando estiver em um quadro, crie um novo cartão com um link ou texto copiado usando "Control/Command + V".

Mover Cartão para Lista Adjacente

,.<>

Ao pressionar "," ou ".", o cartão será movido para a parte inferior da lista adjacente à esquerda ou direita. Ao pressionar os sinais de maior e de menor (< e >), o cartão será movido para a parte superior da lista adjacente à esquerda ou direita.

Meu Filtro de Cartões

Q

Pressione a tecla q para alternar o filtro Cartões atribuídos a mim.

Seguir

S

Pressione "s" para seguir ou deixar de seguir um cartão. Siga um cartão para receber notificações sobre as ações relacionadas a ele.

Atribuir a Si Mesmo

Space

Pressionar a barra de espaço atribuirá você ao cartão ou cancelará sua atribuição.

Editar Título

T

Ao exibir um cartão, pressione t para editar o título. Passe o mouse sobre um cartão e pressione t para abrir o cartão e editar o título.

Votar

V

Pressione v para votar em um cartão (ou remover seu voto) se o Power-Up Votar estiver ativado.

Limpar Todos os Filtros

X

Use x para limpar todos os filtros de cartão ativos.

Abrir Página de Atalhos

?

Pressione ? para abrir a página de atalhos.

Preencher membros automaticamente

@

Ao escrever um comentário, você pode digitar @ e o nome de um membro, um nome de usuário ou iniciais e obter uma lista dos membros correspondentes. Você pode navegar pela lista com as setas para cima e para baixo. Se você pressionar Enter ou Tab com um membro selecionado, ele será mencionado no comentário. O usuário mencionado receberá uma notificação logo após o envio.

Ao adicionar um novo cartão, você pode usar o mesmo método para atribuir membros a cartões antes de enviá-los.

Preencher etiquetas automaticamente

#

Ao adicionar um novo cartão, você pode digitar # e a cor ou o título da etiqueta e obter uma lista de etiquetas correspondentes. Você pode usar setas para cima e para baixo a fim de navegar na lista resultante. Pressione Enter ou Tab para adicionar a etiqueta ao cartão redigido. As etiquetas serão adicionadas ao cartão quando você enviá-lo.

Preencher posição automaticamente

^

Ao adicionar um novo cartão, você pode digitar ^ e um nome de lista ou uma posição em uma lista. Também pode digitar “início” ou “fim” para adicionar ao início ou ao fim da lista atual. Você pode usar setas para cima e para baixo a fim de navegar na lista resultante. Pressione Enter ou Tab para alterar automaticamente a posição do cartão redigido.

Copiar cartão

Command/ControlC / V

Quando você passa o cursor sobre um cartão, pressionar Control + C (Windows) ou Command + C (Mac) copiará para sua área de transferência o cartão. Colar pressionando Control + V (Windows) ou Command + V (Mac) enquanto passa o cursor sobre uma lista copiará o cartão para a lista. Isso também funcionará entre quadros diferentes.

Mover Cartão

Command/ControlX / V

Quando você passa o cursor sobre um cartão, pressionar Control + X (Windows) ou Command + X (Mac) copiará para sua área de transferência o cartão. Colar pressionando Control + V (Windows) ou Command + V (Mac) enquanto passa o cursor sobre uma lista moverá o cartão para a lista. Isso também funcionará entre quadros diferentes.

Desfazer ação

Z

Pressione "Z" para desfazer sua última ação em um cartão.

Refazer ação

ShiftZ

Depois de desfazer uma ação, pressione "Shift + Z" para refazer a última ação que foi desfeita.

Repetir ação

R

Ao visualizar ou passar o cursor sobre um cartão, pressione "R" para repetir sua última ação em um cartão diferente.

Rolar a tela para os lados

ShiftScroll

Segure "Shift" enquanto usa a rolagem do mouse para rolar para a esquerda e a direita em um quadro.
