Atalhos de Teclado

Navegar Cartes

/ J / K

Pressione as teclas de seta para selecionar cartes adjacentes em um quadro. Pressione j para selecionar o carto abaixo do atual. Pressione k para selecionar o carto acima do atual.

Navegar pelos versos de cartes

/ K / J

Pressione as teclas de seta para a esquerda e para a direita para abrir cartes adjacentes em uma lista. Pressione "j" ou a seta para a direita para abrir o carto abaixo do atual. Pressione "k" ou a seta para a esquerda para abrir o carto acima do atual.

Abrir/fechar barra lateral

[

Pressione "[" para abrir e fechar o menu da barra lateral de navegao de quadros.

Alternar Menu do Quadro

W / ]

Pressione "w" ou "]" para recolher ou expandir o menu do quadro, a barra lateral direita.

Abrir Menu Quadros de Cabealho

B

Pressione "b" para pesquisar quadros rapidamente. Navegue pelos resultados da pesquisa com as setas para cima e para baixo. Pressione Enter com um quadro selecionado para abri-lo.

Focalizar Caixa de Pesquisa

/

Pressione / para posicionar o cursor na caixa de pesquisa, no cabealho.

Arquivar Carto

C

Pressione c para arquivar um carto.

Data de Entrega

D

Pressione d para abrir um seletor de data de entrega para um carto.

Adicionar checklist

-

Pressione "-" para adicionar uma checklist a um carto.

Anexar link

Ao visualizar um carto, anexe um link copiado usando "Control/Command + V".

Fechar Menu/Cancelar Edio

Esc

Pressione Esc para fechar uma caixa de dilogo ou janela pop-up aberta ou cancelar as edies e os comentrios que voc est escrevendo.

Salvar texto

Command/ControlEnter

Pressionar as teclas Control + Enter (Windows) ou Command + Enter (Mac) salvar o texto que est sendo escrito. Isto funciona ao escrever e editar comentrios, ao editar o ttulo do carto, o ttulo da lista, a descrio e outras coisas.

Abrir Carto

Pressione Enter para abrir o carto selecionado no momento. Pressione Shift+Enter ao enviar um carto para abri-lo imediatamente aps sua criao.

Pressione m para abrir o menu Adicionar/Remover Membros. Clique no avatar de um membro para atribu-lo ou cancelar sua atribuio.

Inserir Novo Carto

N

Pressionar "n" abre uma janela pop-up que permite adicionar um carto depois do que est selecionado, ou em uma lista vazia.

Criar novo carto com texto

Quando estiver em um quadro, crie um novo carto com um link ou texto copiado usando "Control/Command + V".

Mover Carto para Lista Adjacente

,.<>

Ao pressionar "," ou ".", o carto ser movido para a parte inferior da lista adjacente esquerda ou direita. Ao pressionar os sinais de maior e de menor (< e >), o carto ser movido para a parte superior da lista adjacente esquerda ou direita.

Preencher membros automaticamente

@

Ao escrever um comentrio, voc pode digitar @ e o nome de um membro, um nome de usurio ou iniciais e obter uma lista dos membros correspondentes. Voc pode navegar pela lista com as setas para cima e para baixo. Se voc pressionar Enter ou Tab com um membro selecionado, ele ser mencionado no comentrio. O usurio mencionado receber uma notificao logo aps o envio.

Ao adicionar um novo carto, voc pode usar o mesmo mtodo para atribuir membros a cartes antes de envi-los.

Preencher etiquetas automaticamente

#

Ao adicionar um novo carto, voc pode digitar # e a cor ou o ttulo da etiqueta e obter uma lista de etiquetas correspondentes. Voc pode usar setas para cima e para baixo a fim de navegar na lista resultante. Pressione Enter ou Tab para adicionar a etiqueta ao carto redigido. As etiquetas sero adicionadas ao carto quando voc envi-lo.

Preencher posio automaticamente

^

Ao adicionar um novo carto, voc pode digitar ^ e um nome de lista ou uma posio em uma lista. Tambm pode digitar incio ou fim para adicionar ao incio ou ao fim da lista atual. Voc pode usar setas para cima e para baixo a fim de navegar na lista resultante. Pressione Enter ou Tab para alterar automaticamente a posio do carto redigido.

Colar pressionando Control + V (Windows) ou Command + V (Mac) enquanto passa o cursor sobre uma lista copiar o carto para a lista. Isso tambm funcionar entre quadros diferentes.
